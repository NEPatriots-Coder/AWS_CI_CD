Infrastructure & Deployment

This portfolio is built using a modern cloud-native architecture on AWS, orchestrated with Terraform for infrastructure as code (IaC). The static assets are hosted in an S3 bucket configured for website hosting, while CloudFront serves as a content delivery network (CDN) to ensure low-latency access globally.

Key components:

  • S3 Bucket: Configured with website hosting enabled, using bucket policies for secure access
  • CloudFront Distribution: Edge-optimized content delivery with HTTPS enforcement and custom domain configuration
  • Route53: DNS management for custom domain routing
  • Terraform: Infrastructure as code for reproducible deployments and version control

The entire infrastructure is version controlled alongside the application code, enabling consistent deployments across environments and automated CI/CD pipelines.
